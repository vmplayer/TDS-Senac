# Exercícios Extras — Arrays em JavaScript

## Exercício 7 — Procurando uma Linguagem

Crie o array:

```js
let linguagens = [
    "JavaScript",
    "Python",
    "Java",
    "C#",
    "PHP"
];
```

Pergunte ao usuário qual linguagem ele deseja pesquisar.

Utilize:

- `includes()`

Mostre:

```txt
Linguagem encontrada!
```

ou

```txt
Linguagem não encontrada!
```

---

## Exercício 8 — Posição do Produto

Crie o array:

```js
let produtos = [
    "Mouse",
    "Teclado",
    "Monitor",
    "Webcam",
    "Notebook"
];
```

Pergunte ao usuário qual produto deseja localizar.

Utilize:

```js
indexOf()
```

Exemplo:

```txt
Produto encontrado na posição 2.
```

Caso não exista:

```txt
Produto não encontrado.
```

---

## Exercício 9 — Editando uma Lista

Crie o array:

```js
let cores = [
    "Azul",
    "Verde",
    "Vermelho",
    "Amarelo"
];
```

Substitua:

```txt
Vermelho
```

por

```txt
Roxo
```

utilizando `splice()`.

Depois exiba:

```txt
Azul
Verde
Roxo
Amarelo
```

---

## Exercício 10 — Último Elemento

Crie um array com 5 números.

Utilize:

```js
pop()
```

para remover o último elemento.

Mostre:

```txt
Array antes:
10, 20, 30, 40, 50

Array depois:
10, 20, 30, 40
```

---

## Exercício 11 — Primeiro Elemento

Crie um array com 5 nomes.

Utilize:

```js
shift()
```

para remover o primeiro nome.

Mostre a lista atualizada usando `for...of`.

---

## Exercício 12 — Adicionando no Início

Crie o array:

```js
let times = [
    "Grêmio",
    "Inter",
    "Juventude"
];
```

Adicione:

```txt
São José
```

na primeira posição usando:

```js
unshift()
```

Mostre o array usando `forEach()`.

---

## Exercício 13 — Cadastro de Idades

Peça ao usuário 5 idades e armazene em um array.

Depois exiba:

```txt
Pessoa 1 → 18 anos
Pessoa 2 → 25 anos
Pessoa 3 → 40 anos
```

Utilizando `forEach()` com índice.

Quantidade de animais cadastrados: 7

.length

## Exercício 15 — Desafio Intermediário

Crie um sistema de músicas.

1. Cadastre 5 músicas.
2. Exiba usando `forEach()`.
3. Pergunte qual música deseja procurar.
4. Utilize `includes()`.
5. Caso seja encontrada, mostre sua posição usando `indexOf()`.
6. Caso não exista, informe ao usuário.

Exemplo:

```txt
Digite a música:
Numb

Música encontrada!
Posição: 3
```

# Desafio Final — Gerenciador de Filmes

Faça um programa que:

1. Cadastre 5 filmes.
2. Exiba todos utilizando `forEach()`.
3. Mostre um menu:

```txt
1 - Adicionar filme
2 - Remover filme
3 - Procurar filme
4 - Listar filmes
5 - Sair
```

4. Utilize:

- `push()`
- `splice()`
- `includes()`
- `indexOf()`
- `forEach()`
- `for...of`

5. O sistema deve continuar executando com um `while` até o usuário escolher:

```txt
5 - Sair
```

Esse desafio reúne praticamente tudo o que foi visto nas aulas de arrays até agora.
