## Ainda não ta completo, se quiser dar uma enroladinha assim pra ver, sabe?

### Se já quiser ver agora:

Abra o jogo utilizando `node .\main.js`, faça as missões e, quando completar elas, pressione `6` para ver as conquistas.

No jogo, você precisará concluir todas as **missões**, elas são também as **conquistas**.

### Inimigos:

#### Sombra

São os filhos diretos Do Criador. Desde que Ele abandonou seus filhos para a morte no mundo, eles ficaram bem mais fracos do que eram no tempo que eram considerados da Realeza.

#### O Mascarado

São criaturas que pertencem ao mundo convencional, fazem parte de uma espécie de seita medonha que sugam a essência das sombras para ficarem mais fortes.

#### O Criador

O Criador desse mundo, ele fica zanzando pelo mundo eliminando todos os forasteiros.

Ele culpa os humanos pelos deuses terem amaldiçoado o seu reino.

Se você se encontrar com ele, **fuja**.

---

### Sobre os itens

Todos os itens de ataque darão exatamente **2 PONTOS** de dano.

Todos os itens de defesa darão exatamente **12 PONTOS** de vida.
