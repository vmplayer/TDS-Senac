## Atividade

**Parte 1 - Requisitos**

1. Liste 5 requisitos funcionais priorizados (use a tabela como guia)
2. Especifique 3 requisitos não funcionais críticos para seu projeto
3. Classifique cada RF com nível de complexidade (baixa/média/alta)

**Parte 2 - Histórias de Usuário**

1. Escreva 3 histórias de usuário com critérios de aceitação completos
2. Identifique para cada história: persona envolvida e valor gerado

**Parte 3 - Prototipagem**

1. Esboce o wireframe da página principal (papel ou digital)
2. Inclua anotações sobre interações importantes
3. Pense na jornada do usuário a partir dessa tela

### Atividade Resolvida - Parte 2

```
Como visitante,
quero poder ver uma galeria de imagens em carrossel do trabalho,
para avaliar a qualidade do serviço.

Critérios de aceitação:

- Dado que estou visualizando a galeria de imagens, quero terminar a visualização e ir para outra visualização.
- As imagens devem permitir a visualização em tela cheia.
- Outras soluções devem estar disponíveis abaixo.

Persona: Visitante (uma pessoa comum que está apenas vasculhando o site).
Valor gerado: Demonstração de confiança e sucesso no trabalho.
```
```
Como interessado,
quero poder ser redirecionado para o meio de contato selecionado,
para marcar um orçamento.

Critérios de aceitação:

- Dado que estou no menu de contato, quero clicar no botão e ser redirecionado para o contato automaticamente.
- Deve conter uma mensagem simples pré-digitada dentro do contato".
- Deve usar um link direto para o contato.

Persona: Interessado (Uma pessoa com algum problema e quer marcar um orçamento).
Valor gerado: Facilita o contato.
```
```
Como desconfiado,
quero poder ver as informações da empresa no rodapé,
para garantir a segurança do meu dinheiro.

Critérios de aceitação:

- Dado que estou dentro do site, quero poder ver o CNPJ da empresa e garantir o bom trabalho.
- O CNPJ deve ficar visível junto ao nome social da empresa.
- As informações devem ser copiáveis (ou seja, com user-select padrão)

Persona: Desconfiado (Uma pessoa que tem medo de sofrer golpes ou jogar seu dinheiro no lixo).
Valor gerado: Garante a existência da empresa e todas as suas legalidades.
```
